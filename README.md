Emergency Rescue Assistant and stepPrep

Emergency Rescue Assistant is a safety prototype built in two versions: a desktop app and a mobile app. It detects possible danger using camera, microphone, and gesture signals, then shows emergency status, saves evidence, and helps the user find nearby shelter options.

The desktop version includes live monitoring, editable go-bag lists, Ukrainian/English localization, automatic nearby metro-shelter search, and Google Maps route opening. The mobile version provides a simplified rescue interface with SOS actions, shelter feed, GPS support, emergency contact dialing, and the same localization system.

The app can search for close metro shelter candidates using geolocation and OpenStreetMap/Overpass, without requiring the user to manually enter coordinates or a Google Maps API key.

Note: the app starts in demo mode and should be tested carefully before any real emergency-call integration is enabled.
