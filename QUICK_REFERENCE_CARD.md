## Q&A Quick Answers

**"How accurate is detection?"**  
Multi-sensor verification (camera + audio + gesture) reduces false positives. Demo mode allows calibration for specific environments.

**"Privacy concerns?"**  
Demo mode default, local storage only, evidence saved only during alarms, user controls all data.

**"Why better than existing systems?"**  
Traditional CCTV needs constant monitoring. StepPrep uses AI for automatic detection. Works with existing cameras. No monthly fees.

**"Works offline?"**  
Core features yes (detection, local alarms, evidence capture). Cloud features no (remote alerts, shelter search).

**"Testing methodology?"**  
Unit tests, integration tests, platform tests (web/mobile/desktop), UX tests, demo mode validation, localization review.

**"Future plans?"**  
More languages, wearable support, smart home integration, medical emergency detection, municipal partnerships.
